# IoT-Based Patient Health Monitoring System

## Overview
This project is an IoT-based system designed to monitor patient health parameters in real time and assist doctors in remote monitoring.

## Features
- Real-time patient monitoring
- Emergency alert system
- Remote access for doctors
- Improves response time in critical situations

## Technologies Used
- Arduino IDE
- IoT

## Working
Sensors collect patient data and send it to the system. The data can be accessed remotely by doctors, allowing continuous monitoring even when medical staff are not physically present.

## Files Included
- Arduino Code (.ino)
- Project Documentation (PDF)

## Outcome
This system enhances patient safety by enabling continuous monitoring and faster response during emergencies.
